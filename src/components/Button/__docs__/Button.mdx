# Button

#### Example

<Canvas of={Button.Primary} />

## Usage

```ts
import { Button } from "@erichayesjohnson/ankl";

const Example = () => {
  return (
      <Button
        size="sm"
        text="Button"
        onClick={()=> console.log("clicked")}
        variant="primary"
      />
  );
};

export default Example;
```

```ts
import { Button } from "@erichayesjohnson/ankl";

const ExampleWithStyleOverrides = () => {
  return (
      <Button
        size="md"
        text="Button"
        onClick={()=> console.log("clicked")}
        variant="secondary"
        styles: {
          ...{
            backgroundColor: "pink",
            borderWidth: "3px",
            "&:hover": {
             backgroundColor: "lightgreen",
            },
          },
        },
      />
  );
};

export default ExampleWithStyleOverrides;
```

#### Arguments

- **text** - A string that represents the text content of the button.
- **variant** - A string of either "primary" or "secondary", "primary being the default.
- **disabled** - A boolean indicating whether the button should be disabled or not.
- **size** - A string with one of three possible values: "sm," "md," or "lg." It defines the size or dimensions of the button.
- **onClick** - A function that is called when the button is clicked. It receives a MouseEventHandler for handling the click event on the button element.
- **styles** - A CSS style object. Custom style overrides that can be applied to the button.
